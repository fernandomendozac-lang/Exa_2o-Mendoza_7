# MyDigitalShield - Gestión de Seguridad Personal

**MyDigitalShield** es una aplicación web desarrollada en Java que permite a los usuarios administrar de forma centralizada sus activos digitales, niveles de riesgo y métodos de autenticación. El proyecto combina la gestión técnica de datos con educación sobre legislación y tips de ciberseguridad.

## Características
- **Gestión de Activos (CRUD):** Alta, consulta, edición y borrado de registros de seguridad.
- **Análisis de Riesgo:** Clasificación de servicios según su criticidad (Bajo, Medio, Alto).
- **Control de Autenticación:** Registro de métodos MFA (Multi-Factor Authentication) y fechas de rotación de claves.
- **Sección Informativa:** Consejos sobre administración de seguridad y marco legal (Protección de datos).

## Tecnologías Utilizadas
* **Lenguaje:** Java 8
* **Tecnología Web:** Java Server Pages (JSP) & Servlets
* **Base de Datos:** MySQL (vía XAMPP)
* **Interfaz:** HTML5, CSS3 y Bootstrap 5
* **Servidor:** Apache Tomcat 8.x/9.x
* **Patrón de Diseño:** MVC (Modelo-Vista-Controlador)

## Requisitos del Sistema
Para ejecutar este proyecto localmente, necesitarás:
1.  **XAMPP** (con servicios Apache y MySQL activos).
2.  **NetBeans IDE 8.2** o superior.
3.  **JDK 1.8** configurado.
4.  **Connector/J** de MySQL (archivo `.jar`) en la carpeta de librerías del proyecto.
